### Option 1: Quick Setup
```bash

```

### Option 2: Requirements File
Create a `requirements.txt` file:
```text
pandas>=1.3.0
numpy>=1.21.0
matplotlib>=3.4.0
seaborn>=0.11.0
scikit-learn>=1.0.0
scipy>=1.7.0
```

Then install:
```bash
pip install -r requirements.txt
```
