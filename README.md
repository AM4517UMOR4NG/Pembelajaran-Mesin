## 🚀 Quick Start

### Prerequisites
```bash
✅ Python 3.7+
✅ Jupyter Notebook
✅ Basic understanding of Python
```

### 🏃‍♂️ Setup in 3 Steps
```bash
# 1. Clone the repository
git clone <https://github.com/AM4517UMOR4NG/Pembelajaran-Mesin.git>
cd "Mechanical Lesson"

# 2. Create virtual environment (recommended)
python -m venv ml_env
source ml_env/bin/activate  # Linux/Mac
# ml_env\Scripts\activate   # Windows

# 3. Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn scipy yellowbrick
```

```bash
jupyter notebook
```

## 🔧 Installation Guide

### Option 1: Quick Setup
```bash
pip install pandas numpy matplotlib seaborn scikit-learn scipy yellowbrick
```

### Option 2: Requirements File
Create a `requirements.txt` file:
```text
pandas>=1.3.0
numpy>=1.21.0
matplotlib>=3.4.0
seaborn>=0.11.0
scikit-learn>=1.0.0
scipy>=1.7.0
yellowbrick>=1.4.0
jupyter>=1.0.0
```

Then install:
```bash
pip install -r requirements.txt
```

### Option 3: Conda Environment
```bash
conda create -n ml_course python=3.8
conda activate ml_course
conda install pandas numpy matplotlib seaborn scikit-learn scipy jupyter
pip install yellowbrick
```

## 🐛 Troubleshooting

### Common Issues:
<details>
<summary><strong>📦 Import Errors</strong></summary>

**Problem:** `ModuleNotFoundError: No module named 'sklearn'`
**Solution:** 
```bash
pip install scikit-learn
```
</details>

<details>
<summary><strong>📊 Dataset Not Found</strong></summary>

**Problem:** Dataset file not found
**Solution:** 
- Ensure dataset files are in the correct folder
- Check file paths in the notebooks
- Download missing datasets from provided sources
</details>

<details>
<summary><strong>🎨 Visualization Issues</strong></summary>

**Problem:** Plots not displaying properly
**Solution:** 
```python
%matplotlib inline
import matplotlib.pyplot as plt
plt.style.use('default')
```
