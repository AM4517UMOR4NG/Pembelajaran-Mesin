HTML TagsDescription
<html>The root element that wraps all content in an HTML document.
<head>Contains meta-information about the document, like title and links to stylesheets.
<body>The main content of the webpage, where visible elements like text, images, and links are placed.
<div>A block-level container used to group other elements together for styling or layout purposes.
<p>Defines a paragraph of text.
<a>Defines a hyperlink to another webpage or resource.
<img>Used to display an image on the webpage.
<ul>Defines an unordered (bulleted) list.
<ol>Defines an ordered (numbered) list.
<li>Defines a list item in either an ordered or unordered list.
<strong>Defines important text, usually displayed in bold.
<em>Defines emphasized text, usually displayed in italics.
<h1>Defines the most important header, usually displayed in large text.
<h2>Defines the second most important header, smaller than <h1> but still large.
<span>Defines a section of text that can be styled or formatted inline.
